# ValueBridge Frontend

React + Vite frontend for the ValueBridge business valuation and marketplace platform.

## Setup

1. Install dependencies: `npm install`
2. Create `.env.local` with `VITE_API_URL=http://127.0.0.1:8000`
3. Run: `npm run dev`
4. Open: `http://localhost:5173`
